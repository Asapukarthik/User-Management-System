## 🌟 Key Features

### 🔐 Advanced Security & Auth
- **JWT-Based Authentication**: Secure login mechanism via JSON Web Tokens.
- **Role-Based Access Control (RBAC)**: Enforced authorization across three scopes (`Admin`, `Manager`, and `User`).
  - *Admins* have complete system control.
  - *Managers* can oversee standard users but cannot access Admin identities.
  - *Users* have restricted access to modify only their own data.
- **Robust API Hardening**: Integration of `helmet` for HTTP headers, express-validator, and advanced error handling.

### 👥 Comprehensive User Management
- **Soft Deletion Mechanism**: Users are never permanently deleted by default. Soft deletion flags accounts while archiving previous emails to allow future reclamation.
- **Audit Logging**: Tracking of identity creation, modification, and restoration attached securely inside the database (`createdBy`, `updatedBy`).
- **Approval Workflows**: Configurable requirements for newly provisioned user approvals.

### 💻 Beautiful UI/UX
- **Premium SaaS Dashboard**: Beautiful, responsive layout styled with Tailwind CSS v4 featuring glassmorphism elements, custom HSL color properties, and dynamic interactions.
- **Real-Time Validations**: Frontend form checking synced powerfully with backend error handling.
- **Lucide Icons & Responsive Design**: Lightweight, crisp SVG icons optimized seamlessly for Desktop, Tablet, and Mobile views.

## 📦 Local Installation & Setup

Ensure you have [Node.js](https://nodejs.org/) and [MongoDB](https://www.mongodb.com/) installed on your machine.

### 1. Clone the repository
```bash
git clone <your-repo-link>
cd User-Management-System
```

### 2. Setup the Backend
Navigate to the backend directory and install dependencies:
```bash
cd backend
npm install
```

Create a `.env` file in the `/backend` matching your local environment variables:
```env
PORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/user-management
JWT_SECRET=your_super_secret_key_here
```

Seed the initial Admin account (Optional but recommended):
```bash
npm run seed
```

Start the backend server:
```bash
npm run dev
```

### 3. Setup the Frontend
Open a new terminal window, navigate to the frontend directory, and install dependencies:
```bash
cd frontend
npm install
```

Start the Vite development server:
```bash
npm run dev
```

The frontend should now be running at `http://localhost:5173`. You can log in using the credentials generated by your seeder.
